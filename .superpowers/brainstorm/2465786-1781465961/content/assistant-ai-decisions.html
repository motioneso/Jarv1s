Assistant & AI: Decisions

Main settings should tune Jarvis. Technical provider setup exists, but behind an advanced path.

Default page shape
Assistant profile
Name, tone, directness, humor, recovery/accountability style.
AI source summary
Shared assistant or personal provider. Status phrased as ready / needs setup.
Add provider
Single button opens a flow: pick supported provider, then fill guided fields.
Advanced AI setup
Provider/model rows, model IDs, base URLs, credentials, CLI subscription setup.
Module tool controls
Not on the default page. Link out to module settings where tool permissions belong.

Cut from this default page

Capability lookup, assistant tool list, raw provider catalog, and model-routing internals. These can exist as diagnostics or per-module settings, but not as normal settings content.
