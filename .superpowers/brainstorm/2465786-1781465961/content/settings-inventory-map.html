Settings Surface: Current Inventory

This is a product map of what the current core platform settings page contains, plus the major backend settings surfaces that are not yet exposed in the UI.

Current /settings page
Profile
Email, name, role. Read-only identity summary.
Connectors
Provider catalog, connected accounts, manual Google setup, revoke/status controls.
AI Configuration
Providers, credentials, models, capabilities, routing lookup, assistant tool list.
Admin Identity
Auth provider status, users, pending approvals, registration toggles.
Host / Ops
Live chat multiplexer selection and local host availability.
Missing from UI
Module enablement, audit events, memory controls, task prefs, notification sensitivity.

Personal Settings
