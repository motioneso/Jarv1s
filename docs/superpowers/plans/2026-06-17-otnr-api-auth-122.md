# OTNR API/Auth 122 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close the live, safe subset of OTNR #122 by returning 404 for no-match chat memory-fact mutations and moving bootstrap audit insertion behind a narrow SECURITY DEFINER function.

**Architecture:** Keep all product access through `DataContextDb` and existing route/repository boundaries. For chat facts, have repository mutations return an affected-row boolean via `RETURNING id`, then map false to the existing 404 shape. For bootstrap audit, add a new idempotent settings-owned SQL migration that creates a narrow `app.record_bootstrap_owner_audit_event(...)` SECURITY DEFINER helper owned by `jarvis_migration_owner`, then expose it only through the settings package public API for `bootstrap_owner_created`.

**Tech Stack:** TypeScript, Fastify inject integration tests, Kysely SQL templates, PostgreSQL RLS/SECURITY DEFINER migrations, Vitest.

- [ ] **Step 4: Add new idempotent SQL migration**

Create `packages/settings/sql/0095_bootstrap_audit_security_definer.sql`:

```sql
DROP POLICY IF EXISTS admin_audit_events_bootstrap_insert ON app.admin_audit_events;
CREATE POLICY admin_audit_events_bootstrap_insert ON app.admin_audit_events
  FOR INSERT TO jarvis_migration_owner
  WITH CHECK (
    action = 'bootstrap_owner_created'
    AND target_type = 'user'
    AND actor_user_id = target_id::uuid
    AND metadata = jsonb_build_object('recordedBy', 'record_bootstrap_owner_audit_event')
  );

CREATE OR REPLACE FUNCTION app.record_bootstrap_owner_audit_event(
  actor_user_id uuid,
  target_user_id uuid,
  request_id text
)
RETURNS void
LANGUAGE sql
SECURITY DEFINER
SET search_path = app, pg_temp
AS $$
  INSERT INTO app.admin_audit_events (
    id,
    actor_user_id,
    action,
    target_type,
    target_id,
    metadata,
    request_id,
    created_at
  )
  VALUES (
    gen_random_uuid(),
    actor_user_id,
    'bootstrap_owner_created',
    'user',
    target_user_id,
    jsonb_build_object('recordedBy', 'record_bootstrap_owner_audit_event'),
    request_id,
    now()
  );
$$;

ALTER FUNCTION app.record_bootstrap_owner_audit_event(uuid, uuid, text)
  OWNER TO jarvis_migration_owner;

REVOKE EXECUTE ON FUNCTION app.record_bootstrap_owner_audit_event(uuid, uuid, text) FROM PUBLIC;
GRANT EXECUTE ON FUNCTION app.record_bootstrap_owner_audit_event(uuid, uuid, text)
  TO jarvis_app_runtime;
```
