`@jarv1s/*` **subpath** specifiers at all (it treats everything after `@jarv1s/` as a literal

null → edge silently dropped, invisible to the walker). A **bare** `@jarv1s/*` specifier (no
subpath) resolves via that package's `package.json` `exports["."]` — i.e. the WHOLE barrel —
and the walker is blind to type-only vs runtime exports (regex matches both `import` and
`export` identically).

`packages/shared/src/*` (a package barrel-reachable from every module's web bundle via
