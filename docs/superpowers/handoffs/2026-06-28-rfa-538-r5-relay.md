## Critical Security Invariants (NEVER skip)

- `normalized_value` is private — strip from ALL REST responses and tool outputs
- `source_ref` is private — only appears in `getIndexingState` (worker-only)
- `people.merge` + `people.splitIdentity`: `risk: "destructive"`, executionPolicy MUST NOT be `"auto"`
- `people.acceptMatch`: throws `RequiresExplicitActionError` for `merge_people`/`split_identity` candidates
- Job payloads: metadata only — `actorUserId, source, sourceRefHash, sourceVersion?, reason, idempotencyKey`
- `assertMetadataOnlyPersonPayload` called at top of every worker handler
- Memory sync in SEPARATE transaction from person-context writes
